## Что в коде в этой папке соответствует концепции

| Идея из Notion | Где в репозитории |
|----------------|-------------------|
| Русскоязычный тёплый собеседник, не «бот-ассистент» | `prompts/base_metapelet.txt` + `build_system_prompt()` в `app.py` |
| Персонализация под человека | `prompts/profiles/*.txt` + optional `profile_id` в POST `/chat` |
| Голос | Маршрут `/tts` — Google TTS (`GOOGLE_API_KEY` в окружении) |
| Простой веб-вход | `index.html` — анкета → затем голосовой чат |

Стек: Flask, Anthropic API (Claude Haiku), переменные через `.env` (ключи не коммитить).
